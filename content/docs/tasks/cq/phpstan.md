[PHPStan](https://phpstan.org) finds bugs in your code without writing
tests.

This task will run PHPStan on any of your PHP code to ensure there are no
issues.

### Example Configuration

```yaml
# githaxs_settings/ghx.yml

phpstan:
  installation:
    org: true
   ```
